# Imo Harmonia

O Imo Harmonia e uma landing page informativa sobre o projeto Imo Harmonia - Urbanismo Descomplicado.

## O que e

Uma plataforma com foco em Inteligencia Artificial aplicada ao urbanismo, criada para simplificar a interpretacao de regulamentos urbanisticos e PDM.

## Para que serve

- Explicar o problema da complexidade regulamentar no urbanismo.
- Apresentar a proposta de valor do Imo Harmonia para arquitetos, engenheiros e tecnicos.
- Comunicar o contexto academico do projeto no Instituto Politecnico da Guarda.
- Destacar o reconhecimento no Poliempreende 2025.

## Resumo

Este site funciona como pagina de apresentacao do projeto, mostrando a visao, funcionalidades e impacto esperado do Imo Harmonia.
